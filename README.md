# students_performance_insights-project
Python-based data analytics project analyzing student marks to identify top subjects and performers with visual insights.
Analyzed student performance dataset using Python (Pandas, Matplotlib, Seaborn) to identify the correlation between study hours, attendance, and academic scores. Generated actionable insights and visualizations to predict key factors affecting performance.

Tools Used:
Python, Pandas, Matplotlib, Seaborn, Google Colab

Analyzed student marks across multiple subjects.
Found highest scoring subjects using Pandas group-by functions.
Visualized average marks using bar charts.
